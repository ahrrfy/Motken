## ✅ ما هو ممتاز في النظام

| الجانب | التقييم |
|--------|---------|
| عزل المساجد | ✅ محكم — كل مسجد لا يرى بيانات غيره |
| حماية الأدوار | ✅ الطالب لا يستطيع رفع صلاحياته |
| نظام النقاط | ✅ محمي — فقط المعلم/المشرف/الأدمن |

| صفحة التبرعات | ✅ صفحة معلوماتية — لا تحتاج API |
| Self-healing | ✅ تنظيف جلسات + بيانات منتهية تلقائياً |

### الأمر 2 — إضافة `/api/students`
```
في server/routes.ts
ابحث عن أول app.get("/api/users"
وقبله أضف مباشرة:

app.get("/api/students", requireAuth, async (req, res) => {
  try {
    const currentUser = req.user!;
    let students;
    if (currentUser.role === "admin") {
      students = await storage.getUsersByRole("student");
    } else if (currentUser.mosqueId) {
      students = await storage.getUsersByMosqueAndRole(currentUser.mosqueId, "student");
    } else {
      students = [];
    }
    res.json(students.filter(s => !s.pendingApproval));
  } catch (err: any) {
    res.status(500).json({ message: "حدث خطأ" });
  }
});

أعطني تأكيداً بعد الإضافة.
```
